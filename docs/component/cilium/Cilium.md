# 什么是Cilium

基于eBPF的网络插件，具有良好的可观测性和安全性。

Cilium是基于eBPF的新内核技术，能够在Linux中动态插入指令，从而可以达到安全，数据跟踪，数据控制等业务。因为eBPF运行在内核，因此可以在不修改业务代码情况下应用和跟新安全策略。

特点：

- 网络

  - 原生支持服务负载均衡和出口管理
  - 可扩展的Kubernetes CNI，对数据量和时延大大增强
  - 支持多集群的链接

- 可观察性

  - 身份感知
  - 高级自助服务观察
  - 网络指标和策略故障排查

- 安全性

  - 透明加密
  - 取证和审计
  - 高级网络策略
